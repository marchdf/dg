\subsection{Description}\label{scalar__def_8h_Description}
This function communicates the elements which are not in my partition. Basically you send the elements of U that other partitions need and you receive the elements from other partitions that you will need. You store these in the back columns of U.

Had to declare Nfields when we operate on just one field instead of N\-\_\-\-F
