Parameters

[in]outputformatted solution from the printer class

Description

This outputs the time step, the particle position, and the average solution in the element containing the particle. A more accurate output would interpolate the solution at the particle position (instead of averaging) but that's complicated (see Lget_velocity_at_position function).
