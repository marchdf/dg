Description

This function builds all the interfaces. For each interface, it will find both elements on the right and left. If it's on a boundary, it will use the BC information to find the correct neighboring element.

For MPI processes, there a bunch of conditions to take care of so things might get a bit convoluted. Hang on to your hats.
